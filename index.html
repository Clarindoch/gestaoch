📖 Manual do Sistema
Gestão CH v6 · Guia completo
🏠 Dashboard
Cards clicáveis filtram a lista de clientes · 🚨 Urgentes = 15+ dias sem visto · 📝 Incompletos = sem CPF/doc/endereço · 👁️ Controle de Vistos = clique 👁️ para marcar sem abrir a ficha · 📅 Calendário = azul=aulas, vermelho=GRU · 📌 Notas = post-its coloridos
👥 Clientes
CH 🏢 = Minha empresa → código CH001, CH002... | CTR 🏆 = Clube → CTR001... | EXT 🔖 = Avulso → sem código
Chips de filtro rápido · Busca por nome/CPF/código · Dedup automático ao abrir
📋 Painel do Cliente (6 abas)
⚡ Resumo = nome, CPF, senha, serviços, GRUs | 👤 Dados = ficha + criar pasta Drive | 🔧 Serviços = processos + GRU | 🔫 Armas = ilimitadas (clube e filiação aqui) | 📄 Decl. = 3 por linha, gerar individual ou em lote | 📁 Docs = organizar documentos por serviço
☁️ CORS — Erro "Failed to fetch"
Apps Script → Implantar → Gerenciar implantações → ✏️ Editar → Acesso: "Qualquer pessoa" → Criar nova versão → Implantar → Copiar a nova URL → colar em Configurações
💾 Configurações persistem
URL do GAS, Token, ID da Planilha e Pasta Drive ficam salvos no localStorage. Quando atualizar o sistema, as configurações são recuperadas automaticamente. Se sumirem, vá em Configurações e salve novamente.
