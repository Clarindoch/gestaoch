Gestão CH v2.0 · Offline First
🏠 Dashboard
Stats: CH, CTR, Ativos, Inativos, GRUs a pagar, Urgentes, Incompletos, Alunos
Calendário: Clique em qualquer dia para ver aulas e GRUs
Alertas: Visto 13-14 dias = ⚠️ Atenção · 15+ dias = 🚨 Urgente
Ações rápidas: Cadastro rápido, novo cliente, novo aluno, relatório
👥 Clientes
Cadastro Rápido ⚡: Só nome + telefone/CPF opcional — alerta de incompleto automático
Novo Cliente ➕: Cadastro completo com todos os campos
👁️ Marcar Visto: Registra a data do último contato — controla prazo
⚠️ Incompleto: Badge laranja indica campos faltando — clique para completar
Status de serviços: Pendente Cliente · Pendente Sistema · Pronto p/ Análise · Em Análise · Deferido · Indeferido
💰 GRU
Gerar: Dentro do popup do serviço → botão 💰 GRU
Vencimento: Automático em 30 dias — editável
✅ Pagar: Marca como paga com data · ↩ Desfazer: Volta para A Pagar
🎓 Alunos & Cursos
Modalidades: Básico, Avançado 1, Avançado 2 — incompatíveis no mesmo dia
Regras: Somente sáb/dom · máx 4 alunos/dia · 1 curso por dia
🔒 Bloquear dia: Laudo, Feriado, Evento — nenhum aluno entra no dia
Google Agenda: Ao cadastrar, escolha Sim para abrir e criar o evento
Vincular cliente: Se o aluno já é cliente, vincule para cruzar os dados
📊 Relatório
Selecione clientes → 📝 para obs individual → 📄 Gerar PDF
O arquivo HTML gerado pode ser aberto no navegador e impresso como PDF
💾 Dados & Sync
Offline: Todos os dados ficam no celular/PC — funciona sem internet
Sync: Configure a URL do Google Apps Script em Configurações para sincronizar
Backup: Configurações → Exportar Dados (arquivo JSON)
Restaurar: Configurações → Importar Dados
🔄 Sync Google Sheets
Configure para sincronizar os dados com o Google Sheets quando estiver online.
